Objetivos

Los objetivos del curso son los siguientes:

  • Separar contenido y presentación de capa web.
  • Conocer los nuevos tags introducidos con HTML5.
  • Conocer los nuevos selectores y atributos introducidos con CSS3.
  • Aprender lo básico de JavaScript y JQuery.

  • Aprender a escribir CSS de impresión.
  • Aprender a utilizar técnicas de responsive design.
  • Aprender a utilizar otras tipografías más allá de las consideradas web-safe.
  • Conocer los nuevos patrones de interfaces de usuario.

  • Diseñar páginas web responsive que se adapten a cualquier dispositivo.

  • Escribir selectores de CSS más eficientes, mejorando los tiempos de renderizado del navegador.
  • Emplear iconos mediante font-face reduciendo las peticiones al servidor.
  • Diagnosticar el render de una página web en cualquier navegador de clase A.
  • Introducir nuevos tags HTML5 en aplicaciones existentes o de nuevo desarrollo, con soporte para navegadores más antiguos.
  • Utilizar schema.org en sus propios proyectos.
  • Utilizar Modernizr y polyfills para soportar navegadores sin soporte de HTML5.
  • Aplicar frameworks CSS y CSS reset.
  • Administrar sus hojas de estilo mediante un pre-procesador de CSS (Sass, LESS, Stylus).
  • Validar la accesibilidad del HTML.
  • Optimizar el ancho de banda consumido por las aplicaciones web.
  • Emplear las herramientas de ayuda al desarrollo que hay disponibles online.
  • Realizar un SEO básico de sus aplicaciones web.
  • Identificar el grado de soporte de cada navegador.
  • Realizar pruebas a la vez en distintas versiones de IE.
  • Usar LiveReload para editar su página web sin necesidad de recargar en el navegador.

Curso Oficial Spring Core

Curso de 4 días de duración en el cual los asistentes aprenderán el funcionamiento del core framework de Spring para Dependency Injection. Al finalizar este curso los asistentes serán capaces de desarrollar aplicaciones de negocio de una forma ágil y escalable, fácilmente adaptables a la evolución de las necesidades futuras. Tras este curso el asistente dispondrá de la posibilidad de certificarse con Spring realizando gratuitamente el Spring Framework Certification Examination online (cuyo valor de mercado es 115 euros).

El curso ofertado no se centra únicamente en la explicación de las características del framework y su correcta utilización, sino que además se tratarán temas de arquitectura de sistemas.

Es importante resaltar que los asistentes al curso deben poseer conocimientos previos de los APIs de Java, conceptos JEE, JTA y del servlet API.

Spring Core es el entrenamiento perfecto si:

  • Se está interesado en todo lo relacionado sobre el desarrollo de aplicaciones Java con Spring
  • Se comienza un proyecto próximamente y se necesita aprender Spring rápidamente
  • Desea mejorar su currículum obteniendo la certificación oficial de Spring

Para continuar mejorando los conocimientos de Spring es recomendable asistir a Spring Web o Enterprise Integration with Spring una vez completado Spring Core.

Objetivos

Al final del curso:

  • Tendrás conocimiento del lenguaje de programación Groovy y el framework Grails
  • Conocerás todo lo necesario para el desarrollo de aplicaciones, desde la capa web hasta la de persistencia
  • Entenderás las mejores prácticas claves para el desarrollo de aplicaciones de alta calidad

Al completar este curso

Serás capaz de hacer lo siguiente:

  • Usar Groovy para desarrollar aplicaciones.
  • Integración con Java.
  • Usar Groovy para añadir funcionalidad dinámicamente.
  • Usar Grails para el desarrollo web y la persistencia.
  • Asegurar la calidad de tus aplicaciones Grails a través de tests unitarios y de integración.
  • Usar los mecanismos de scaffolding básicos para construir aplicaciones robustas.
