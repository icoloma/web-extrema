Casos de éxitoSúmate a la lista de clientes que han confiado en nuestros servicios de consultoría y mentoring.

  • Aena Navegación Aérea Durante este proyecto de un año se formó un equipo de 16 personas sin experiencia previa, en un proyecto de desarrollo de un sistema de mensajería aérea en tiempo real que integra más de 10 fuentes diferentes de datos con recuperación de errores.

  • Sistema de control de tráfico marítimo Durante este proyecto de seis meses se formó desde cero un equipo para desarrollar un producto de control de tráfico marítimo con alertas (colisión, zonas de peligro, etc) e integración de sistemas de detección en tiempo real (radares, video, goniómero, etc) y back-end noSQL.

  • Arquitectura para desarrollo de aplicaciones Durante este proyecto de un año se actualizó la arquitectura para aplicaciones en sistemas distribuidos de Banco de España a un sistema basado en arquitectura Spring. El equipo de este proyecto incluyó seis arquitectos, dos de ellos de la entidad.

  • Plataforma de E-banking Desarrollo de plataforma de e-banking para la rama de banca privada de BBVA. Este proyecto incluye el desarrollo completo de un front-end web que da acceso a los productos contratados por el usuario, incluyendo informes en tiempo real de las posiciones de portfolio. El producto final incluye la integración de una compleja plataforma de seguridad basada en sistemas de seguridad de última generación.

  • Training en Tecnologías Web La Oficina del Alto Comisionado para los Derechos Humanos de las Naciones Unidas en Ginebra ha recibido una sesión intensiva de training en tecnologías HTML5 y CSS3, para dar un mejor soporte a sus páginas Intranet y Extranet.
