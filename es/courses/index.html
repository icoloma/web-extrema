HTML5 y CSS3

Este curso introduce las últimas modificaciones en HTML5 y CSS3, dedicando especial atención al soporte en Firefox, Chrome e Internet Explorer.

Durante tres días revisaremos las mejores prácticas de desarrollo web actualizadas a los nuevos estándares y se analizarán patrones y técnicas actualizadas de diseño contrastadas con las versiones antiguas. Éste es un curso muy completo que cubre desde wireframing y patrones de capa de presentación hasta una breve introducción a SEO y distribución del mercado de navegadores, incluyendo una detallada descripción de las versiones de IE que siguen siendo relevantes.

Curso Oficial Core Spring

SpringSource es la empresa que soporta Spring, plataforma que con más de cinco millones de descargas se ha convertido en estándar de facto para las aplicaciones empresariales basadas en tecnología Java. Más de la mitad de las empresas que constituyen el Fortune 500, el 60% de los desarrolladores Java y 9 de cada 10 bancos utilizan la plataforma de SpringSource.

El curso oficial de Spring Core introduce todos los conceptos, desde básico a avanzado, relativo al uso del framework Spring. Desde inyección de dependencias, aspectos o desarrollo de interfaces REST con seguridad integrada, este curso constituye una introducción obligada para poder desarrollar aplicaciones con Spring.

Enterprise Integration con Spring (NUEVO)

Durante este curso de cuatro jornadas dirigidos por expertos en Spring se hará un recorrido por las tecnologías de mayor relevancia para la integración de aplicaciones en un entorno empresarial.
