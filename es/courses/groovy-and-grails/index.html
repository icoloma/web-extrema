Groovy & Grails

Duración:
32 horas
Idioma:
Este curso está disponible en Inglés e Italiano.
Dirigido a:
Desarrolladores, arquitectos
Requisitos:
Experiencia desarrollando aplicaciones Java

Objetivos

Al final del curso:

  • Tendrás conocimiento del lenguaje de programación Groovy y el framework Grails
  • Conocerás todo lo necesario para el desarrollo de aplicaciones, desde la capa web hasta la de persistencia
  • Entenderás las mejores prácticas claves para el desarrollo de aplicaciones de alta calidad

Al completar este curso

Serás capaz de hacer lo siguiente:

  • Usar Groovy para desarrollar aplicaciones.
  • Integración con Java.
  • Usar Groovy para añadir funcionalidad dinámicamente.
  • Usar Grails para el desarrollo web y la persistencia.
  • Asegurar la calidad de tus aplicaciones Grails a través de tests unitarios y de integración.
  • Usar los mecanismos de scaffolding básicos para construir aplicaciones robustas.
