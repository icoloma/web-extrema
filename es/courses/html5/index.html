Acepto

Volver a cursos

HTML5 y CSS3

Duración:
Tres jornadas de ocho horas
Idioma:
Este curso está disponible en español e inglés.
Dirigido a:
Diseñadores, maquetadores de HTML y desarrolladores de código servidor (java, python, ruby) de capa web. Es recomendable un mínimo de un año desarrollando aplicaciones web (HTML y CSS). No son necesarios conocimientos de JavaScript.
Requisitos:
Cada alumno deberá llevar su propio portátil, para el que deberá tener permisos de administrador. Se recomiendan los sistemas operativos Windows (XP, Vista, 7) o distribuciones Linux, dado que son los sistemas con los que el instructor tendrá experiencia. Las herramientas de software que sean necesarias se irán instalando a lo largo del curso. En cursos in-house, es imprescindible que la sala de impartición del curso disponga de conexión a Internet para todos los asistentes.

Objetivos

Los objetivos del curso son los siguientes:

  • Separar contenido y presentación de capa web.
  • Conocer los nuevos tags introducidos con HTML5.
  • Conocer los nuevos selectores y atributos introducidos con CSS3.
  • Aprender lo básico de JavaScript y JQuery.
  • Aplicar los nuevos componentes de formulario y reglas de validación presentes en HTML5.
  • Asumir la cuota de mercado de cada navegador, con tratamiento especial para las versiones de IE.
  • Simplificar el código HTML generado por el servidor.
  • Utilizar frameworks de CSS (Twitter Bootstrap, Compass, Less)
  • Reducir el tráfico entre navegador y servidor, mejorando el rendimiento.
  • Mejorar la accesibilidad de las aplicaciones desarrolladas.
  • Optimizar el posicionamiento en buscadores.
  • Mantener una imagen coherente entre navegadores, incluyendo aquellos que no soportan HTML5 o CSS3.
  • Aprender a escribir CSS de impresión.
  • Aprender a utilizar técnicas de responsive design.
  • Aprender a utilizar otras tipografías más allá de las consideradas web-safe.
  • Conocer los nuevos patrones de interfaces de usuario.

Al completar este curso

Cuando termine este curso el asistente será capaz de:

  • Crear e interpretar wireframes eficientemente.
  • Desarrollar aplicaciones web basadas en HTML5 y CSS3.
  • Sustituir código JavaScript por animaciones basadas en CSS3, más rápidas y eficientes en recursos.
  • Diseñar páginas web responsive que se adapten a cualquier dispositivo.
  • Emplear y generar sprites para sus diseños.
  • Escribir selectores de CSS más eficientes, mejorando los tiempos de renderizado del navegador.
  • Emplear iconos mediante font-face reduciendo las peticiones al servidor.
  • Diagnosticar el render de una página web en cualquier navegador de clase A.
  • Introducir nuevos tags HTML5 en aplicaciones existentes o de nuevo desarrollo, con soporte para navegadores más antiguos.
  • Utilizar schema.org en sus propios proyectos.
  • Utilizar Modernizr y polyfills para soportar navegadores sin soporte de HTML5.
  • Aplicar frameworks CSS y CSS reset.
  • Administrar sus hojas de estilo mediante un pre-procesador de CSS (Sass, LESS, Stylus).
  • Validar la accesibilidad del HTML.
  • Optimizar el ancho de banda consumido por las aplicaciones web.
  • Emplear las herramientas de ayuda al desarrollo que hay disponibles online.
  • Realizar un SEO básico de sus aplicaciones web.
  • Identificar el grado de soporte de cada navegador.
  • Realizar pruebas a la vez en distintas versiones de IE.
  • Usar LiveReload para editar su página web sin necesidad de recargar en el navegador.
