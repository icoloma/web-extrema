Spring Web

El objetivo de este curso es aprender a utilizar la tecnología Spring para crear aplicaciones web completas. El desarrollo de aplicaciones web basadas en Spring desde cero y de forma escalonada junto con una metodología enfocada a resolver los problemas reales más habituales hacen de este curso un complemento ideal para aquéllos que ya tienen cierta experiencia con la tecnología Spring.

Aunque no es requisito imprescindible, para un mejor aprovechamiento del curso se recomienda haber realizado previamente el curso Spring Core.

    Duración:
    4 jornadas de 8 horas
    Idioma:
    Este curso está disponible en español, inglés e italiano.
    Dirigido a:
    Desarrolladores Java y arquitectos implicados en proyectos web.
    Requisitos:
    Cada asistente deberá llevar su propio portátil, en el que deberá tener permisos para instalar aplicaciones.

    Objetivos

      Contenido

      1. Spring MVC Día 1
        • Modelo de programación basado en anotaciones Spring MVC
        • Composición de páginas con tecnologías basadas en Layouts como Tiles
        • Visualización de multiples tipos de contenidos
        • Gestión de Excepciones
        • Procesado de formularios
      2. Enriqueciendo el interfaz de usuario Día 2
        • Buenas prácticas para el enriquecimiento del interfaz del usuario.
        • Fundamentos de JavaScript para desarrollo Ajax
        • Uso del toolkit de Dojo y Spring JS
        • Internacionalización y personalización
      3. Spring Web Flow Día 3
        • Introducción a Spring WebFlow. Fundamentos del lenguaje de definición de flujos.
        • Uso de Actions para invocar lógica de negocio
        • Gestión de eventos Ajax, Excepciones y reglas de navegación dinámicas.
      4. Aplicaciones Robustas y Seguras Día 4
        • Spring Faces. Integración con JSF
        • Construcción de pruebas funcionales
        • Protección de aplicaciones web con Spring Security
        • Uso de Grails y Roo para el desarrollo rápido de aplicaciones web.
