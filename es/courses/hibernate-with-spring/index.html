Hibernate con Spring (NUEVO)

En este curso de 3 días aprenderás cómo crear aplicaciones data-driven efectivas con Spring y Hibernate. En el curso se enseña tanto la configuración usando XML como la configuración basada en anotaciones de Spring y Hibernate.

Con nuestros cursos te beneficiarás de la enorme experiencia de nuestro equipo. Te traemos esta experiencia en un curso altamente interactivo y práctico.

    Duración:
    24 horas
    Idioma:
    Este curso está disponible en Ingés e Italiano.
    Dirigido a:
    Este training esta dirigido a desarrolladores interesados en el utilizo de Hibernate con Spring.
    Requisitos:
    Haber creado aplicaciones web con Java; Conocimiento básico de JDBC y bases de datos relacionales

    Objetivos

    Tras finalizar este curso, obtendrás conocimiento sobre la creación de aplicaciones basadas en Spring y Hibernate y serás capaz de:

    • Configurar Hibernate en el contexto de una aplicación basada en Spring
    • Crear eficientemente mapeos object/relational con Hibernate
    • Utilizar Spring para facilitar las sesiones de Hibernate, así como las transacciones y el manejo de excepciones
    • Utilizar Hibernate con Spring y comprender temas de arquitectura fundamentales
    • Saber cómo y cuándo utilizar partes de Spring Framework

    Contenido

    1. Spring Framework Foundations
      • Introducción a Spring
      • Usar Spring para configurar una aplicación
      • Simplificar la configuración de aplicaciones
      • Inyección de dependencias basada en anotaciones
      • Simplificar el acceso a datos basado en JDBC con Spring
      • Manejar transacciones de bases de datos usando Spring
    2. Hibernate ORM
      • Introducción a object/relational mapping (ORM)
      • Empezar con Hibernate en un entorno con Spring
      • Utilizar anotaciones JPA y XML de Hibernate
      • Herencia: consideraciones y "trade-offs"
      • Definir tipos propios como Value Objects
      • Usar consultas: HQL, criteria API, nativo (SQL)
      • Hibernate en aplicaciones web: abrir sesión en la vista
      • Utilizar los diferentes mapeos de collecciones Java
      • Usar joins y componentes que profundizan en el mapeo de la clase a la tabla
    3. Hibernate Internals y configuración
      • Trabajar con los objetos de Hibernate
      • Lazy loading y detached objects
      • Bloqueo optimista y pesimista
      • Mezclar Hibernate y JDBC
      • Performance tuning
      • Características avanzadas: interceptores, generación del schema, cacheo, batching
