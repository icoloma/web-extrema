Curso Oficial Core Spring

Curso de 4 días de duración en el cual los asistentes aprenderán el funcionamiento del Core Spring framework Spring de Dependency Injection. Al finalizar este curso los asistentes serán capaces de desarrollar aplicaciones de negocio de una forma ágil y escalable, fácilmente adaptables a la evolución de las necesidades futuras. Tras este curso el asistente dispondrá de la posibilidad de certificarse con Spring realizando gratuitamente el Spring Framework Certification Examination online (cuyo valor de mercado es 115 euros).

El curso ofertado no se centra únicamente en la explicación de las características del framework y su correcta utilización, sino que además se tratarán temas de arquitectura de sistemas. Para continuar mejorando los conocimientos de Spring es recomendable asistir a Rich Web Applications with Spring o Enterprise Integration with Spring una vez completado Core Spring.

Es importante resaltar que los asistentes al curso deben poseer conocimientos previos de los APIs de Java, conceptos JEE, JTA y del servlet API.

    Duración:
    4 jornadas de 8 horas
    Idioma:
    Este curso está disponible en español, inglés e italiano.
    Dirigido a:
    Desarrolladores Java y arquitectos.
    Requisitos:
    Cada asistente deberá llevar su propio portátil, en el que deberá tener permisos para instalar aplicaciones.

    Objetivos

      Contenido

      1. Fundamentos de Spring Framework Día 1
        • Introducción a la arquitectura de contenedor ligero y el Spring Application Context
        • Desarrollo de aplicaciones estructuradas, altamente configurables y testables a través de Dependency Injection y Test-Driven Development.
        • Desarrollo de código mejor y más cohesivo, con debate sobre los patrones y mejores prácticas de diseño orientado a objetos.
        • Realización de prácticas basadas en modelos de negocios reales.
      2. Arquitectura eficiente de capa de negocio Día 2
        • Programación orientada a objetos pragmática en la capa de negocio, con ejemplos y sin tecnicismos
        • Persistencia efectiva: cómo construir una capa de persistencia con Spring utilizando las mejores prácticas.
        • Análisis y balance de las estrategias relevantes de persistencia, incluyendo Hibernate y Spring JDBC
        • Estrategias de gestión de transacciones con Spring en la capa de servicios
        • Estrategias de test unitarios y de integración en la capa de negocio
      3. Implementando conectividad con información Empresarial Día 3
        • Análisis de conectividad ORM e Hibernate con Spring
        • Visión general de Spring Web MVC, Spring JavaScript, Spring Web Flow y Spring Faces
        • Configuración de servicios remotos gestionados por Spring
        • Diseño de los primeros contratos de servicios web a través de Spring
      4. Integración con Enterprise Services Día 4
        • Introducción a Spring Security y el acceso basado en roles
        • Gestión a través de Spring JMX
        • Configuración de Message-Driven POJOs con Spring JMS
        • Análisis de las soluciones empresariales de SpringSource
