Case studies

  • AenaAir Navigation Messaging system During this 1-year project a team with no prior experience with Java was formed to develop a real-time messaging system for air navigation messages that connects 10+ data sources with error recovery mecanisms.
  • Vessel Traffic System A team was created from scratch for this six-month project to develop a system that could control the maritime traffic of a city in Europe, including 20+ different types of navigation alarms (hazards, colisions, restricted areas) and the integration of multiple detection systems in real time (radar, video, goniometer..). The platform integrates a NoSQL data store with a multithreading Java platform, and is actually scaling up to control the entire coast of an European country.
  • Architecture for application development During this one-year project, we modified the architecture for application development of distributed systems to migrate existing systems to a Spring-based platform. This project included a team of six architects, two of them from the bank.
  • E-banking platform Development of the e-banking platform for the private banking branch of BBVA. This project includes the full development of a web front end that enables access to the user assets, including real-time reports of portfolio positions. The final product includes the integration of a complex security platform based on state-of-the-art authentication systems in the back-end.
  • Web technologies training The IT team of the Office of the High Commissioner of Human Rights of the United Nations has received an intensive series of training courses on HTML5 and CSS3 technologies to better support their Intranet and Extranet sites.
