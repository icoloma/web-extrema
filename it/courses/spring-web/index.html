Spring Web

Spring Web è un corso di formazione di 4 giorni per lo sviluppo di applicazioni web. In questo corso gli studenti impareranno come applicare, in modo pratico, le ultime novità in Spring per creare applicazioni web.

Il completamento di questa formazione dà diritto ad ogni studente di ricevere un voucher gratuito per pianificare un esame in un Pearson VUE Center per diventare un Vmware® SpringSource® Certified Spring Web Application Developer.

Spring Web é il training ideale se:

  • Vuoi diventare un esperto nell&qout;utilizzo di Spring Framework
  • Vuoi mostrare le tue capacità di sviluppatore costruendo ottime web application
  • Hai già seguito il corso Core Spring e vuoi migliorare ulteriormente le tue capacità di sviluppatore
  • Stai lavorando su/stai pianificando un progetto web
  • Sei interessato a ricevere la certificazione Spring Web

Per migliorare ulteriormente le tue capacità di sviluppatore Spring ti suggeriamo di realizzare Enterprise Integration with Spring dopo aver completato Spring Web.

Durata:
32 ore
Lingua:
Il corso é in italiano ma in modalità in-house può essere impartito anche in inglese.
A chi é rivolto:
E’ quindi importante sottolineare che gli alunni devono possedere previa conoscenza della tecnologia Spring.
Requisiti:
Ogni alunno deve portare con sé un laptop ed avere un administrator account. Sono preferibili Windows (XP, Vista, 7) o Linux, con i quali l'istruttore ha maggior familiarità. Altri software tool saranno installati durante i laboratori. Nei corsi in modalitá in-house, é necessaria una connessione Internet per tutti.

Obiettivi

Al termine del training avrai una buona comprensione di Spring e delle tecnologie associate per lo sviluppo web e sarai capace di:

  • sviluppare applicazioni web usando Spring Framework
  • Usare Spring Web Flowper implementare interazioni di tipo stateful
  • Rendere sicure le applicazioni web con Spring Security
  • Testare le applicazioni web per correttezza e performance
  • Creare interfacce utente di tipo rich web con HTML, CSS e JavaScript
  • Connettere un cliente Flex ad un back-end Spring con Spring BlazeDS
  • Mettere tutto insieme con rapidi strumenti di produttività come Grails e Spring Roo
  • Usare SpringSource Tool Suite e Spring Insight

Contenuto

  1. Applicando MVC Giorno 1
    • Visione generale del corso e delle applicazioni corrispondenti. Aspetti base di Spring MVC.
    • Integrazione di varie tecnologie e formati per l’esposizione dei contenuti, includen do JSP/JSTL, Tiles, PDF ed Excel.
    • Data-Entry Form, Binding e Validazione.
  2. Implementazione di Interfacce di tipo Rich Giorno 2
    • Analisi delle migliori tecniche per il disegno di interfacce utente in relazione al con tenuto, lo stile ed il comportamento delle stesse.
    • Aspetti fondamentali di Javascript per sviluppatori Ajax.
    • Massimizzazione delle funzionalitá del Dojo Toolkit.
    • Internazionalizzazione, personalizzazione ed accessibilità di Applicazioni Web sviluppate con Spring.
  3. Utilizzo di Spring Web Flow Giorno 3
    • Primi passi con Spring Web Flow.
    • Aspetti essenziali del linguaggio utilizzato nella definizione dei Flow (Flow Definition Language). Stati, Eventi e Transizioni. Aggiungere comportamenti ad un Flow.
    • Controllo di eventi Ajax, gestione dinamica delle eccezioni e delle regole di navigazione.
  4. Sviluppo di applicazioni web affidabili e sicure Giorno 4
    • Integrazione di componenti JSF con Spring Faces.
    • Test di accettazione e di carico durante l’intero ciclo di sviluppo di un Applicazione Web.
    • Hibernate in contesti di operatività su grandi volumi ed alta concorrenza.
    • Integrazione di vari sistemi di applicazioni, includendo servizi gestiti da Spring, OSGi e Servizi Web.
