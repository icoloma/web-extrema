Corso Ufficiale Core Spring

    Durata:
    32 ore
    Lingua:
    Il corso é in italiano ma in modalità in-house può essere impartito anche in inglese.
    A chi é rivolto:
    E’ quindi importante sottolineare che gli alunni devono possedere previa conoscenza delle API Java, dei concetti JEE, JTA e delle API Servlet.
    Requisiti:
    Ogni alunno deve portare con sé un laptop ed avere un administrator account. Sono preferibili Windows (XP, Vista, 7) o Linux, con i quali l'istruttore ha maggior familiarità. Altri software tool saranno installati durante i laboratori. Nei corsi in modalitá in-house, é necessaria una connessione Internet per tutti.

    Obiettivi

      Contenuto

      1. Fondamenti di Spring Framework Giorno 1
        • Introduzione all’ architettura di tipo lightweight, IoC Container ed Application Context.
        • Sviluppo di applicazioni strutturate, configurabili e testabili attraverso Dependency Injection e Test-Driven Development.
        • Sviluppo di codice migliore e maggiormente coeso, con utilizzo di Design Patterns ed Object Oriented.
        • Realizzazione di esercizi basati su modelli di business reali.
      2. Business Layer Giorno 2
        • Aggiungere comportamenti agli oggetti con l’ AOP (Aspect Oriented Programming).
        • Introduzione alla Persistenza: come costruire il Persistence Layer con Spring.
        • Persistenza con Spring JDBC.
        • Strategie di gestione delle transazioni con Spring nel Service Layer.
        • Strategie di test d’unità e di integrazione a livello di Business Layer.
      3. Enterprise Information Connectivity Giorno 3
        • Introduzione all’ ORM (Object-to-Relational Mapping).
        • Utilizzo e configurazione di Hibernate con Spring.
        • Architettura di una Web Application con Spring.
        • Introduzione a Spring Web MVC, Spring JavaScript, Spring Web Flow e Spring Faces.
        • Spring MVC.
        • RESTful Web Services con Spring MVC.
      4. Integrazione con servizi Enterprise Giorno 4
        • Rendere sicure le applicazioni con Spring Security.
        • Introduzione a Spring Remoting.
        • Come semplificare applicazioni JMS tramite Message-Driven POJOs con Spring JMS.
        • Gestione delle applicazioni con JMX tramite Spring.
