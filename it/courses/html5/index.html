Torna ai corsi

HTML5 e CSS3

Durata:
24 ore
Lingua:
Il corso é in italiano ma in modalità in-house può essere impartito anche in inglese.
A chi é rivolto:
Designer e sviluppatori web server side (java, python, ruby). E' consigliabile aver maturato un'esperienza minima di un anno nello sviluppo di applicazioni web (HTML e CSS). Non é richiesta alcuna esperienza in Javascript.
Requisiti:
Ogni alunno deve portare con sé un laptop ed avere un administrator account. Sono preferibili Windows (XP, Vista, 7) o Linux, con i quali l'istruttore ha maggior familiarità. Altri software tool saranno installati durante i laboratori. Nei corsi in modalitá in-house, é necessaria una connessione Internet per tutti.

Obiettivi

Lo scopo del corso é:

  • Separare contenuto e presentazione
  • Introdurre i nuovi tag disponibili nell'HTML5..
  • Introdurre i nuovi selettori disponibili nel CSS3.
  • Introdurre le basi di JavaScript e JQuery.
  • Applicare i new form elements e le regole di validazione di HTML5.
  • Comprendere il market share dei browser, specialmente rispetto ad Internet Explorer.
  • Imparare come semplificare il proprio codice HTML.
  • Introdurre vari framework CSS (Twitter Bootstrap, Compass).
  • Migliorare le performance riducendo il traffico tra browser e server.
  • Migliorare l'accessibilità.
  • Ottimizzare il posizionamento nei motori di ricerca.
  • Mantenere un'immagine omogenea includendo legacy browser senza il supporto HTML5 o CSS3.
  • Imparare a scrivere stylesheet CSS alternativi per la stampa.
  • Imparare ad applicare responsive design.
  • Introdurre nuove fonti oltre a web safe.
  • Imparare nuovi pattern UI per web design.

Al termine del corso

Dopo questo corso, l'alunno sarà in grado di:

  • Creare e comprendere i wireframe.
  • Sviluppare applicazioni web basate su HTML5 e CSS3.
  • Sostituire le animazioni JavaScript-based con transizioni CSS3 più rapide ed efficienti.
  • Sviluppare una responsive web page compatibile con ogni dispositivo.
  • Creare ed usare le proprie immagini sprite.
  • Scrivere selettori CSS efficienti che aiutino a migliorare il rendering time.
  • Usare font-face icon per minimizzare le server request.
  • Diagnosticare la propria pagina web in qualunque browser di classe A.
  • Introdurre nuovi tag HTML5 in applicazioni nuove o già esistenti senza rompere la compatibilità browser.
  • Usare schema.org nei propri progetti.
  • Usare Modernizr e polyfills per il supporto con i browser legacy.
  • Usare framework CSS e reset CSS.
  • Gestire i propri stylesheet usando un preprocessor CSS (Sass, LESS, Stylus).
  • Validare l'accessibilità e la struttura del proprio HTML.
  • Ottimizzare la larghezza della banda consumata dalle proprie applicazioni web.
  • Usare tool di sviluppo online.
  • Dare i primi passi con il SEO.
  • Identificare il supporto browser per i nuovi standard.
  • Provare versioni differenti di Internet Explorer allo stesso tempo.
  • Usare LiveReload per editare la propria pagina web in maniera più efficiente.
