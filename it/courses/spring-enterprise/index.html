Enterprise Integration with Spring

Enterprise Integration with Spring è un corso di formazione di quattro giorni guidato da esperti SpringSource che discuteranno i temi più importanti e le tecnologie legate all&qout;Enterprise Application Integration. Questa pratica formazione vi insegna come applicare le tecnologie Spring per risolvere i problemi di integrazione che devono affrontare gli sviluppatori lato server.

Il completamento di questa formazione dà diritto ad ogni studente di ricevere un voucher gratuito per pianificare un esame in un Pearson VUE Center per diventare uno SpringSource Certified Spring Enterprise Integration Specialist.

Enterprise Integration with Spring è il corso ideale se:

  • Vuoi diventare un esperto nell"utilizzo di Spring Framework
  • Hai già realizzato il corso Core Spring e vuoi migliorare ulteriormente le tue capacità di sviluppatore
  • Vuoi essere in grado di risolvere anche i problemi più difficili con Spring
  • Stai lavorando su/stai pianificando un lungo progetto Spring
  • Ti interessa ottenere lacertificazione Spring expert

Per migliorare ulteriormente le tue capacità di sviluppatore Spring ti raccomandiamo di realizzare Spring Web dopo aver completato Enterprise Integration with Spring.

Durata:
32 ore
Lingua:
Il corso é in italiano ma in modalità in-house può essere impartito anche in inglese.
A chi é rivolto:
Esperienza nello sviluppo di applicazioni Java, conoscenza base di Spring e del Container IOC (Inversione di Controllo). Nonostante non sia imprescindibile, é raccomandabile aver realizzato previamente il corso di Core Spring.
Requisiti:
Ogni alunno deve portare con sé un laptop ed avere un administrator account. Sono preferibili Windows (XP, Vista, 7) o Linux, con i quali l'istruttore ha maggior familiarità. Altri software tool saranno installati durante i laboratori. Nei corsi in modalitá in-house, é necessaria una connessione Internet per tutti.

Obiettivi

Alla conclusione del training avrai una buona comprensione di Spring e delle tecnologie associate per l"enterprise integration e sarai capace di:

  • Usare Spring per la creazione di applicazioni concorrenti e la pianificazione dei Task.
  • Usare Remoting per implementare applicazioni client-server.
  • Usare Spring Web Services per la creazione, nel rispetto del principio di Loose Coupling, di Servizi Web e Client basati su SOAP.
  • Usare Spring MVC per la creazione di Servizi Web e Client RESTful.
  • Usare JMS per la gestione asincrona di comunicazioni Messaging-based.
  • Capire e usare transazioni locali e distribuite.
  • Usare Spring Integration per la creazione di architetture Event-driven e Pipes-and-Filters e come integrarle con sistemi ed applicazioni esterne.
  • Usare Spring Batch per la gestione e la scalabilità d’esecuzione di procedure batch basate su componenti standard o create su misura.
